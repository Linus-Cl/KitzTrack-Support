# KitzTrack – Support-Seite (GitHub Pages)

Diese Dateien werden über GitHub Pages veröffentlicht (Ordner `docs/`).

## Einmalig aktivieren

1. GitHub Repo öffnen → **Settings** → **Pages**
2. **Build and deployment**
   - **Source**: *Deploy from a branch*
   - **Branch**: `main`
   - **Folder**: `/docs`
3. Speichern und kurz warten (1–3 Minuten)

Danach ist die Seite unter einer URL wie dieser erreichbar:

- `https://<username>.github.io/<repo>/`

## App Store Connect

- **Support-URL**: `https://<username>.github.io/<repo>/`
- **Datenschutz-URL**: `https://<username>.github.io/<repo>/privacy.html`

## Anpassen

- Ersetze Platzhalter wie `support@DEINE-DOMAIN.DE`, Name/Adresse im Impressum und Datenschutz.
